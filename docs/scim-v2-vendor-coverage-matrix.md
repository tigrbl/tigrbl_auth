# SCIM v2 Vendor Coverage Matrix

Research date: 2026-05-28

Scope: SCIM v2.0 provisioning coverage across `tigrbl_auth`, Auth0, Okta, Keycloak, FusionAuth, and Amazon Cognito. This matrix distinguishes real SCIM protocol support from adjacent user-management APIs.

## Legend

| Marker | Meaning |
|---|---|
| ✅ | Present as a documented SCIM v2 product/API capability. |
| 🟡 | Partial, adjacent, gated, experimental, or supported through a non-SCIM product surface. |
| 🚧 | Planned/underframed in `tigrbl_auth`; helper or governance work exists but no product SCIM API yet. |
| ❌ | No first-class SCIM v2 support found for that vendor/product surface. |
