# Strategic Security Certification Concern Plan

This planning artifact records the initial SSOT evidence target for the strategic
security certification concern ladder. The associated tests are intentionally
planned and skipped until each concern is implemented in a dedicated runtime
slice.

Near-term certification targets:

- Realm isolation certification
- Tenant isolation certification
- Issuer confusion resistance
- Runtime capability truth
- Authorization mutation and freshness semantics
- JOSE negative security
- Cryptographic boundary governance
- Resource server verifier certification
- Machine identity governance
- Delegation provenance
- Authorization observability
- Runtime qualification and deployment truth

Later roadmap targets:

- Constraint-native authorization
- Authorization budgets
- Cross-issuer trust domains
- Credential privacy and correlation controls
- Future standards nonblocking roadmap

The future standards list is tracked as valid but non-blocking for current
maturity: VC/OIDC4VC, OIDC4VCI, OID4VP, SIOPv2, HAIP, Authz API 1.0, OIDC 1.1,
RISC, CAEP, SET, and SSE.
