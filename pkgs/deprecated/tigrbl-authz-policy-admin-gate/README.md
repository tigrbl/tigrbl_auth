# tigrbl-authz-policy-admin-gate

Standalone AdminGate package for the Tigrbl auth package suite.

- Package: `tigrbl-authz-policy-admin-gate`
- Import root: `tigrbl_authz_policy_admin_gate`
- Owns `AdminGate` and the OpenAPI admin security dependency constants.
- `tigrbl-authz-policy.admin_gate`, `tigrbl_identity_server.security.admin_gate`, and `tigrbl_auth.security.admin_gate` are compatibility wrappers.

```python
from tigrbl_authz_policy_admin_gate import AdminGate
```
