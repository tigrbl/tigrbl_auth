# @tigrbl-auth/demo-hub-uix

Demo hub UIX for presenting Tigrbl Auth product surfaces as one cohesive identity system.

Local development:

```bash
npm run dev -w @tigrbl-auth/demo-hub-uix
```

Docker deployment:

```bash
docker compose -f docker/docker-compose.demo-hub-uix.yml up -d --build
```

Default URL: <http://localhost:3020>
